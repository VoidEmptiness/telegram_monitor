Настройка Telegram Monitor
  1. API ID и Hash

    Получите на сайте my.telegram.org. Необходимы для авторизации приложения.

  2. Канал

    Укажите @username канала или полную ссылку на канал.

  3. Ключевые слова

    Введите слова для поиска в сообщениях канала. Можно добавить несколько.

  4. ID личного чата

    Ваш Telegram ID для получения уведомлений.
